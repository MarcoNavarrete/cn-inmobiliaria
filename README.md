# CN Inmobiliaria

Frontend de CN Inmobiliaria para publicar casas, terrenos, desarrollos inmobiliarios y oportunidades de inversion.

## Scripts disponibles

### `npm start`

Ejecuta la aplicacion en modo desarrollo.

### `npm test`

Ejecuta la suite de pruebas en modo interactivo.

### `npm run build`

Genera el build de produccion en la carpeta `build`.
